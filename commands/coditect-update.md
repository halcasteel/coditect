# CODITECT Update

Update CODITECT to the latest version.

## Automatic Execution

Run this command to update CODITECT:

```bash
/opt/coditect/update.sh
```

This will:
1. Check for available updates
2. Download and install the latest version
3. Reset permissions
4. Show update summary

## Usage

Just run `/coditect-update` - no arguments needed.
